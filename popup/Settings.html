Settings for Custom Emoji Addiction

Json syntax must be 
        {
          "{name}": "{imageUrl}"
        }
like 
        {
          "danger_zone": "https://emoji.slack-edge.com/T039B0HP2/danger_zone/5be61f0c1aad398c.png"
        }
You can reference this emoji by using the name in colons like :danger_zone:
The plugin will then go and replace it with picture you have added but as an emoji.
You can also have alias or additional names for the same emojis they have a syntax like 
        {
          "{name1}": "{imageUrl}",
          "{name2}": "alias:{name1}"

        }
This way when you type either :name1: or name2 it will use the same emojis. and example would be 
        {
          "danger_zone": "https://emoji.slack-edge.com/T039B0HP2/danger_zone/5be61f0c1aad398c.png",
          "archer": "alias:danger_zone"
        }
